# 🔐 CyberSecurity - Password & Username Generator

An interactive, terminal-based utility built in Python designed to generate secure, custom-length passwords and random usernames. Featuring sleek ASCII art menus and built-in error handling to ensure seamless execution.

## 🚀 Features
* **Password Generator:** Creates strong, randomized passwords blending uppercase letters, lowercase letters, and numbers.
* **Username Generator:** Instantly generates random, customizable usernames.
* **Batch Generation:** Choose exactly how many passwords or usernames you want to generate in a single run.
* **Interactive Interface:** Clean CLI terminal menu with strict input validation to handle typos gracefully.

## 🛠️ Technologies & Modules Used
* **Python 3**
* **`random` module** (Python standard library)
